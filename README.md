# 家教账单 APP

专为家教老师设计的课程账单管理应用。

## 功能特性

- 📚 **学生管理** - 添加、编辑、删除学生信息
- 📝 **课程记录** - 记录每次上课的日期、课时、金额
- 💰 **收款标记** - 一键标记课程是否已收款
- 📊 **账单统计** - 图形化展示学生账单汇总
- 💾 **本地存储** - 数据保存到手机本地，无需网络

## 使用方法

### 安装依赖
```bash
npm install
```

### 启动项目
```bash
# 启动开发服务器
npm start

# 运行 Android
npm run android

# 运行 iOS
npm run ios

# 运行 Web
npm run web
```

## 技术栈

- React Native + Expo
- TypeScript
- SQLite (expo-sqlite)
- React Navigation
